# App icon & splash (mrapp)

Same as main app: use Kuremedi branding for store builds.

| File | Purpose | Recommended size |
|------|---------|------------------|
| `images/icon.png` | App icon | **1024×1024** px |
| `images/favicon.png` | Web favicon | 48×48 |
| `images/splash-icon.png` | Splash screen | ~200–400 px width |
| `images/android-icon-foreground.png` | Android adaptive (foreground) | 1024×1024 |
| `images/android-icon-background.png` | Android adaptive (background) | 1024×1024 |
| `images/android-icon-monochrome.png` | Android monochrome | 1024×1024 |

Copy from `app/assets/images/` (Kuremedi assets) or export from `website/public/Kure.png` at the sizes above.
